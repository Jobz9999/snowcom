# snowcom

# 概要
このアプリは、スノースポーツの楽しさを共有できるwebアプリケーションです。ユーザーは、訪れた場所、感想、写真や動画
を投稿し、他のユーザーとの情報交換をすることができます。さらに、

## サイトテーマ
「スノースポーツのスキルアップと、情報のシェア」

## テーマを選んだ理由
私は、スキー、スノーボードを趣味にしていますが、スキー場の情報交換やスキルアップのために相談できる環境が少ないと感じていました。
現存のSNSに投稿することもできますが、多くの投稿に埋もれてしまいがちです。スキル別やウインタースポーツに特化して情報収集できる仕組みを
提供することができます。
