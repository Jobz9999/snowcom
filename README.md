# snowcom

# 概要
このアプリは、スノースポーツの楽しさを共有できるwebアプリケーションです。ユーザーは、訪れた場所、感想、写真や動画
を投稿し、他のユーザーとの情報交換をすることができます。LINEのオープンチャットのように、共通の趣味を軸にしたオープンなコミュニティを作成・参加し、
投稿を通じて交流できることを目的としています。

## サイトテーマ
「スノースポーツのスキルアップと、情報のシェア」

## テーマを選んだ理由
私は、スキー、スノーボードを趣味にしていますが、スキー場の情報交換やスキルアップのために相談できる環境が少ないと感じていました。
現存のSNsは投稿を起点としているのに対し、共通の興味（スノーボード、スキー、ゲレンデ、ギア）ごとにユーザーが集まる構造を採用しました。

### [きっかけ]
スノースポーツには、<br>
・初心者と経験者の情報格差が大きい<br>
・ゲレンデ情報・道具・スタイルなど、興味関心が細分化されている<br>
という特徴があります。そのため、「投稿型SNS」ではなく、興味関心ごとに人が集まれるコミュニティを作りたいと考えたことがきっかけです。

### [問題点]
・投稿が流れてしまい、同じゲレンデ・レベル・スタイルの人と出会いにくい<br>
・「スノーボード」「スキー」といった大きな括りでは情報が広すぎる<br>
・一時的な交流で終わり、継続的なコミュニティが形成されにくい<br>

### [解決策]
・スタイル・レベル・ゲレンデ・ギアなどより特化したコミュニティを作成<br>
・ユーザーはコミュニティに参加することで、共通の関心を持つ人と継続的に交流可能<br>
・承認制・モデレーター機能を設け、安心して参加できる運営体制を想定<br>

# ターゲットユーザー
・スノースポーツ好きの20〜50代<br>
・新しいスキー場を探している人<br>
・スキルアップのアドバイスが欲しい人

# 主な利用シーン
・シーズン前に情報収集をしたいとき<br>
・シーズン中のリアルタイム性の高い情報共有・交流<br>
・他のユーザーからアドバイスをもらいたいとき<br>
